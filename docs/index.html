医院について

About Clinic

北かしわ眼科は、JR北柏駅南口すぐの場所にある眼科クリニックです。
令和2年5月に、柏福眼科から名称変更および移転を行い、現在の場所で診療を行っています。

眼科専門医による診療と、視能訓練士（認定視能訓練士を含む）による各種検査を組み合わせ、
患者様一人ひとりに合わせた検査・診断・治療を心がけています。

  • 眼科専門医による診察
  • 視能訓練士による丁寧な検査
  • JR北柏駅から徒歩すぐ
  • 病診連携によるスムーズな紹介

  • 角膜疾患・結膜疾患
  • 白内障

診療案内

Medical Services

北かしわ眼科では、一般眼科診療を中心に、白内障・緑内障・網膜疾患などの診療を行っています。

Shared Decision Making（共同意志決定）を意識し、医師から治療の選択肢を提示した上で、
患者様ご自身に治療内容を選んでいただくことを大切にしています。

受診の流れ

初診の方：
視力検査 → 診察・処置 → 各種検査 → 診察・処置

再診の方：
各種検査（視力を含む）→ 診察・処置

※充血・目やに等があり感染性結膜炎が強く疑われる場合は、先に診察を行うことがあります。
※網膜疾患や白内障がある場合、瞳を開く検査が必要になることがあります。
瞳を開く目薬を使用した後は、4〜5時間程度は車の運転をお控えください。

コンタクトレンズについて

Contact Lenses

現在、コンタクトレンズが初めての方の対応は一時休止しております。
眼鏡処方についても、現在は受付を休止しております。

近視抑制治療薬「リジュセア点眼」が発売されていますが、保険適応外となるため、
診察代・薬代ともに自由診療（10割負担）となります。こども医療費は使用できません。
また、保険診療と自由診療を同日に行うことはできません。

※近視抑制治療のみ予約制（自由診療枠）を設けています。詳細はお問い合わせください。
